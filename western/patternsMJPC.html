Mah Jongg Players Companion Patterns

  1. A Run, A Pung, A Pair: One suit, run 1-9, pung, pair

  2. All Honors: Four Pungs of 1s, 9s, Winds & Dragons, Pair Honors

  3. All Honors Pairs:  Pairs of 1s & 9s only, Winds & Dragons only, or all combined

  4. Big Robert: Three runs of 4 tiles, one in each suit, Pair Winds or Dragons

  5. Chow Chow: 4 Chows and a Pair, all in 1 suit

  6. Dragon’s Tail: Run 1-9, (pair Dragons, pung Winds) OR (pung Dragons, pair Winds)

  7. Dragon’s Teeth:  Run 1-7 OR 2-8 in Craks OR Dots, any tile paired; Pung White, Red Dragons

  8. Five Odd Honors:  Run 1-9, Any 5 Winds or Dragons, No Pairs

  9. Gates Hands

    Confused Gates: 111 in 1st suit, 999 in 2nd suit, 2-8 in 3rd suit with a tile paired

    Gates of Heaven: One suit, 111, 2-8 sequence with a tile paired, 999

    Golden Gates: One suit 22's 44's 66's 88's pairs, pung 111 or 999, pung corresponding Dragon

  10. Gertie’s Garter: Run 1-7 in any 2 suits

  11. Grand Sequence: A 1-9 run, pair any of the run, pair another of the run, Pung Winds/Dragons

  12. Greta’s Dragon: Run 1-7, one each Wind, Ping any Dragon

  13. Greta’s Garden: Run 1-7, one each Wind, One each Dragon

  14. Guardian Dragons: Run 1-9, Pung Dragons, Pair Dragons

  15. Guardian Winds: Run 1-9, Pung Winds, Pair Winds

  16. Hachi Ban: Run 1-8 or 2-9, 3 Pairs Winds or 3 Pair Dragons

  17. Hovering Angel: Chow in each suit, Pung Wind, Pair Dragon

  18. Little Brother: Chow in each suit, Chow any suit, pair Winds

  19. Little Robert: Chow in each suit, Pung any suit, pair any suit

  20. Moon at Bottom of Well: Four Chows and a Pair, all in Dots

  21. Red Lantern: Run 1-7 w/ any tile paired, Pung Reds, Pung (Own) Wind

  22. The Professors: Three mixed Chows, One each Dragon, pair Winds

  23. Three Philosophers: Chow in each suit, Mixed Chow, Pair any suit

  24. Triple Knitting: 4 like numbers with a number in each suit, Pair of numbers with different suit

  25. Windy Chows: Chow each suit, One each Wind, another single any Wind

  26. Wriggly Snake: Run 1-9 , One each Wind, Pair any tile

  27. YinYang: 1,1,2,3,4,5,5 in 1st suit, 5,5,6,7,8,9,9 in 2nd Suit
