MJPC Patterns

  1. A Run, A Pung, A Pair: One suit, run 1-9, pung, pair

  2. All Honors: Four Pungs of 1s, 9s, Winds & Dragons, Pair Honors

  3. All Pair Jade: Six Bam Pairs of 2,3,4,6,8, at least one pair of Green Dragons

  4. All Pair Ruby Jade:  5 Pairs of Red & Green Bams (1 each required), Pair Green Dragons, Pair Red Dragons

  5. All Winds and Dragons:  4 Pungs and a Pair of Winds and Dragons

  6. Apple Blossom: 3 Mixed Chows, Pung White Dragons, Pair Green Dragons

  7. Big Robert: Three runs of 4 tiles, one in each suit, Pair Winds or Dragons

  8. Blue Mountains: 3 Pung and a Pair in 2,3,4,5,8,9 Dot, Pung Green Dragons

  9. Chinese Odds: 4 Pungs and a Pair of Odd Numbers, 1 Suite

  10. Chop Suey: Chow 1,2,3 in each suit, NEWS with 1 paired

  11. Chow Chow: 4 Chows and a Pair, all in 1 suit

  12. Chow Mein: Chow 7,8,9 in each suit, NEWS with 1 paired

  13. Civil War: 1861 in 1st suit, 1865 in 2nd suit, NNN SSS

  14. Confused Gates: 111 in 1st suit, 999 in 2nd suit, 2-8 in 3rd suit with a tile paired

  15. Down You Go: 2222,444,66,8 in 1 suit, 1 each Wind

  16. Dragon's Breath: 5 Pairs in 1 suit, GWR with 1 paired

  17. Dragon's Gates: 1 suit, 2-8 with a tile paired, pung 111 or 999, pung matching dragon

  18. Dragon’s Run: Run 1-9 in 1 suit, 1 each Dragon, Pair Winds

  19. Dragon’s Scales: 3 Pungs and a Pair in 2,4,6,8 Craks, 3 Red Dragons

  20. Dragon’s Tail: Run 1-9, (pair Dragons, pung Winds) OR (pung Dragons, pair Winds)

  21. Dragon’s Teeth:  Run 1-7 OR 2-8 in Craks OR Dots, any tile paired; Pung White, Red Dragons

  22. Dragonette: 3 Pairs in 1 suit, no 1's or 9's, 1 Each Wind, 1 Each Dragon, Pair any Wind or Dragon

  23. Dragonfly:  Pung in each suit, Pair in any suit, 1 each Dragon

  24. Driven Snow: 3 Pungs and a Pair in 1,3,5,7,9 Craks, 3 White Dragons

  25. Five Odd Honors:  Run 1-9, Any 5 Winds or Dragons, No Pairs

  26. Four Blessings: Pung each Wind, Pair of Anything

  27. Gates of Heaven: One suit, 111, 2-8 sequence with a tile paired, 999

  28. Gertie’s Garter: Run 1-7 in any 2 suits

  29. Golden Gates: One suit 22's 44's 66's 88's pairs, pung 111 or 999, pung matching Dragon

  30. Grand Sequence: A 1-9 run, pair any of the run, pair another of the run, Pung Winds/Dragons

  31. Green Jade: 3 Pungs and a Pair in Bams, 3 Green Dragons

  32. Greta’s Dragon: Run 1-7, one each Wind, Ping any Dragon

  33. Greta’s Garden: Run 1-7, one each Wind, One each Dragon

  34. Guardian Dragons: Run 1-9, Pung Dragons, Pair Dragons

  35. Guardian Winds: Run 1-9, Pung Winds, Pair Winds

  36. Hachi Ban: Run 1-8 or 2-9, 3 Pairs Winds or 3 Pair Dragons

  37. Heads and Tails : Four Pungs and a Pair, 1's and 9's only

  38. Heavenly Twins : 7 Pairs, 1 Suit

  39. Hovering Angel: Chow in each suit, Pung Wind, Pair Dragon

  40. Imperial Jade: 3 Pungs and a Pair in Green Bams (2,3,4,6,8), Pung Green Dragons

  41. Knitting: 7 Pairs, 2 Suits, each pair has a matching number in the other suit

  42. Lilly Pilly: 2 Pungs and a Pair in Dots, 3 Green Dragons, 3 White Dragons

  43. Lily of the Valley: 2 Pungs and a Pair in Green Bams (2,3,4,6,8), 3 Green Dragons, 3 White Dragons

  44. Little Brother: Chow in each suit, Chow any suit, pair Winds

  45. Little Robert: Chow in each suit, Pung any suit, pair any suit

  46. Moon at Bottom of Well: Four Chows and a Pair, all in Dots

  47. Numbers Doubled: #2-8 only, 2 Pungs of same number in 2 suits, Pair of Winds or Dragons

  48. Numbers in Parallel: #2-8 only, Pung of same # in 3 suits, Pung and Pair of Winds or Dragons

  49. Odds & Evens: 13579 in 1st and 2nd suit, 2468 in 3rd Suit

  50. Red Coral: 3 Pungs and a Pair in Craks, 3 Red Dragons

  51. Red Lantern: Run 1-7 w/ any tile paired, Pung Reds, Pung (Own) Wind

  52. Red Lily: 2 Pungs and a Pair Red Bams (1,5,7,9), Pung Whites, Pung Reds

  53. Red Waratah: Pung Red Bams (1,5,7,9), Pung Dots, Pung Crak, Pung Reds, Pair Green

  54. Robin: A Chow in 1st suit, 2 Chows in 2nd suit, Chow and Pair in 3rd Suit

  55. Royal Ruby: 3 Pung and a Pair in 1,5,7,9 and Pung Reds

  56. Ruby Jade: Pung in each of Red Bams, Green Bams, Red Dragons, Green Dragons; Pair of Any Bams

  57. Seven Brothers: Pung of 7 in each suit, NEWS with any 1 paired

  58. Sparrow's Sanctuary: 2 pairs 1 Bam, pair of 23468 Bams

  59. Sunrise: Pung in each Suit, #2-8 only, Pung East Wind, Pair White Dragons

  60. Sunset: Pung in each Suit, #2-8 only, Pung West Wind, Pair Red Dragons

  61. The Professors: Three mixed Chows, One each Dragon, pair Winds

  62. Three Philosophers: Chow in each suit, Mixed Chow, Pair any suit

  63. Three Sisters: Pung of 3 in each suit, NEWS with any 1 paired

  64. Triple Knitting: 4 numbers with a number in each suit, Pair of numbers with different suits

  65. Unique Wonder: 1 each 1's, 1 each 9's, 1 each Wind, 1 each Dragon, any of these paired

  66. Up You Go: 2,44,666,8888 in 1 suit, 1 each Wind

  67. White Elephant: 3 Pungs and a Pair in Even Dots, 3 White Dragons

  68. White Opal: 3 Pungs and a Pair in Dots, 3 White Dragons

  69. Windfall: 5 Pairs in 1 suit, 1 Ea Wind

  70. Windvane: Pung in each suit, NEWS with 1 paired

  71. Windy Chow: Chow each suit, One each Wind with 1 paired

  72. Windy Dragons: Pair of each Wind, Pung 2 Dragons

  73. Windy Nines: Pung of each 9, One each Wind, another single any Wind

  74. Windy Ones: Pung of each 1, One each Wind, another single any Wind

  75. Wriggly Dragon: Run 1-9 any suit, 1 EA Dragon, a Pair of Dragons

  76. Wriggly Snake: Run 1-9 , One each Wind, Pair any tile

  77. Yin Yang: 1,1,2,3,4,5,5 in 1st suit, 5,5,6,7,8,9,9 in 2nd Suit
