Mah Jongg Players Companion Patterns

  1. A Run, A Pung, A Pair: One suit, run 1-9, pung, pair

  2. All Honors: Four Pungs of 1s, 9s, Winds & Dragons, Pair Honors

  3. All Honors Pairs:  Pairs of 1s & 9s only, Winds & Dragons only, or all combined

  4. All Pair Jade: Six Bam Pairs of 2,3,4,6,8, or Green Dragons (Must have at least 1 Pair Green Dragons)

  5. All Pair Ruby Jade:  Five Pairs of Bams, Pair Green Dragons, Pair Red Dragons

  6. Big Robert: Three runs of 4 tiles, one in each suit, Pair Winds or Dragons

  7. Blue Mountain: 3 Pung and a Pair in 2,3,4,5,8,9 Dot, Pung Green Dragons

  8. C Odds: 4 Pungs and a Pair of Odd Numbers, 1 Suite

  9. Chow Chow: 4 Chows and a Pair, all in 1 suit

  10. Civil War: 1861 in 1st suit, 1865 in 2nd suit, NNN SSS

  11. Down You Go: 2222,444,66,8 in 1 suit, 1 each Wind

  12. Dragon’s Run: Run 1-9 in 1 suit, 1 each Dragon, Pair Winds

  13. Dragon’s Scales: 3 Pungs and a Pair in 2,4,6,8 Craks, 3 Red Dragons

  14. Dragon’s Snow: 3 Pungs and a Pair in 1,3,5,7,9 Craks, 3 White Dragons

  15. Dragon’s Tail: Run 1-9, (pair Dragons, pung Winds) OR (pung Dragons, pair Winds)

  16. Dragon’s Teeth:  Run 1-7 OR 2-8 in Craks OR Dots, any tile paired; Pung White, Red Dragons

  17. Dragonette: 3 Pairs in 1 suit, no 1's or 9's, 1 Each Wind, 1 Each Dragon, Pair any Wind or Dragon

  18. Dragonfly:  Pung in each suit, Pair in any suit, 1 each Dragon

  19. Five Odd Honors:  Run 1-9, Any 5 Winds or Dragons, No Pairs

  20. Four Blessings: Pung each Wind, Pair of Anything

  21. Gates Hands

    Confused Gates: 111 in 1st suit, 999 in 2nd suit, 2-8 in 3rd suit with a tile paired

    Gates of Heaven: One suit, 111, 2-8 sequence with a tile paired, 999

    Golden Gates: One suit 22's 44's 66's 88's pairs, pung 111 or 999, pung corresponding Dragon

  22. Gertie’s Garter: Run 1-7 in any 2 suits

  23. Grand Sequence: A 1-9 run, pair any of the run, pair another of the run, Pung Winds/Dragons

  24. Green Jade: 3 Pungs and a Pair in Bams, 3 Green Dragons

  25. Greta’s Dragon: Run 1-7, one each Wind, Ping any Dragon

  26. Greta’s Garden: Run 1-7, one each Wind, One each Dragon

  27. Guardian Dragons: Run 1-9, Pung Dragons, Pair Dragons

  28. Guardian Winds: Run 1-9, Pung Winds, Pair Winds

  29. Hachi Ban: Run 1-8 or 2-9, 3 Pairs Winds or 3 Pair Dragons

  30. Heads and Tails : Four Pungs and a Pair, 1's and 9's only

  31. Heavenly Twins : 7 Pairs, 1 Suit

  32. Hovering Angel: Chow in each suit, Pung Wind, Pair Dragon

  33. Imperial Jade: 3 Pungs and a Pair in Green Bams (2,3,4,6,8), Pung Green Dragons

  34. Knitting: 7 Pairs, 2 Suits, each pair has a matching number in the other suit

  35. LillyPilly: 2 Pungs and a Pair in Dots, 3 Green Dragons, 3 White Dragons

  36. Lily of the Valley: 2 Pungs and a Pair in Green Bams (2,3,4,6,8), 3 Green Dragons, 3 White Dragons

  37. Little Brother: Chow in each suit, Chow any suit, pair Winds

  38. Little Robert: Chow in each suit, Pung any suit, pair any suit

  39. Moon at Bottom of Well: Four Chows and a Pair, all in Dots

  40. Numbers in Parallel: Same # in 3 suits, Pung and Pair of Winds/Dragons

  41. Numbers Doubled: #2-8, 2 Pungs each Number, Pair of Winds or Dragons

  42. Odds & Evens: 13579 in 1st and 2nd suit, 2468 in 3rd Suit

  43. Red Coral: 3 Pungs and a Pair in Craks, 3 Red Dragons

  44. Red Lily: 2 Pungs and a Pair Red Bams (1,5,7,9), Pung Whites, Pung Reds

  45. Red Lantern: Run 1-7 w/ any tile paired, Pung Reds, Pung (Own) Wind

  46. Red Waratah: Pung Red Bams (1,5,7,9), Pung Dots, Pung Crak, Pung Reds, Pair Green

  47. Robin: A Chow in 1st suit, 2 Chows in 2nd suit, Chow and Pair in 3rd Suit

  48. Royal Ruby: 3 Pung and a Pair in 1,5,7,9 and Pung Reds

  49. Seven Brothers: Pung of 7 in each suit, NEWS with any 1 paired

  50. Sparrow's Sanctuary: 2 pairs 1 Bam, pair of 23468 Bams (Green Bam)

  51. Sunrise: Pung in each Suit, #2-8 only, Pung East Wind, Pair White Dragons

  52. Sunset: Pung in each Suit, #2-8 only, Pung West Wind, Pair Red Dragons

  53. The Professors: Three mixed Chows, One each Dragon, pair Winds

  54. Three Philosophers: Chow in each suit, Mixed Chow, Pair any suit

  55. Three Sisters: Pung of 3 in each suit, NEWS with any 1 paired

  56. Triple Knitting: 4 numbers with a number in each suit, Pair of numbers with different suits

  57. Unique Wonder: 1 each 1's, 1 each 9's, 1 each Wind, 1 each Dragon, any of these paired

  58. Up You Go: 2,44,666,8888 in 1 suit, 1 each Wind

  59. White Elephant: 3 Pungs and a Pair in Even Dots, 3 White Dragons

  60. White Opal: 3 Pungs and a Pair in Dots, 3 White Dragons

  61. Windfall: 5 Pairs in any suit, 1 Ea Wind

  62. Windvane: Pung in each suit, NEWS with 1 paired

  63. Windy Chows: Chow each suit, One each Wind, another single any Wind

  64. Windy Dragons: Pair of each Wind, Pung 2 Dragons

  65. Windy Ones: Pung of each 1, One each Wind, another single any Wind

  66. Windy Nines: Pung of each 9, One each Wind, another single any Wind

  67. Wriggly Snake: Run 1-9 , One each Wind, Pair any tile

  68. Yin Yang: 1,1,2,3,4,5,5 in 1st suit, 5,5,6,7,8,9,9 in 2nd Suit
